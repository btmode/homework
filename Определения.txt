Свойство - private поле + 2 public метода (getField, setField)
/////
Автосвойство - свойство с дефолтными реализациями методов get и set. 
Дефотлная реализация для get - вернуть значение поля, для set - установить значение в поле
/////
Ссылочные и значимые типы данных
Значимые: int, bool, double, float, структуры
Ссылочные: любые объекты, массивы, словари, строки

Ссылочные сравниваются и копируются по ссылке (кроме строк)
int[] a1 = [1, 2, 3];
int[] a2 = [1, 2, 3];
int[] a3 = a1
a1 == a2 // false
"abc" == "abc" // true

Значимые сравниваются и копируются по значению 
var a = 1;
var b = 1;
a == b // true

Дефолтные значения:
int - 0
bool - false
Для всех ссылочных типов данных значение по умолчанию null
string - null
array - null
object - null



Ref, Out

Ref - позволяет сделать из значимого типа данных ссылочный (то есть передать по ссылке)

Out - другой способ return'a
